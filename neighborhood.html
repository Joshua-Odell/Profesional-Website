Neighborhood

My wife and I recently purchased a Home. The process was daunting. So I decided to make a website that would allow people to get all the relevant information on an address. I used multiple different API's to call different information such as news, school districts and location information.

This project uses a HTML, CSS and JavaScript. The APIs used are Google Maps, Census, School, and Bing News
